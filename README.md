# 🌿 Jungle Escape

### 🎓 High School Capstone Project (Zaključna Naloga)
**Jungle Escape** is a 2D mobile platformer developed for **Android** as my final project in the 4th year of secondary school (Srednja šola). This project demonstrates complete mobile game architecture, custom asset integration, and persistent data management.

---

## 🏆 Project Achievements
* **Perfect Score:** Received **100% (Odlično)** on the final maturity course presentation.
* **Development Time:** Fully developed from concept to deployment over a dedicated **2-month** period.
* **Scope:** Completed a full 10-level loop with all core platformer mechanics implemented from scratch.

---

## 🎮 Gameplay Features

* **Action Platforming:** Intuitive touch-based movement and jumping mechanics tailored for mobile.
* **10 Progressive Levels:** Each level increases in difficulty with new layouts and environmental challenges.
* **Enemy & Hazard System:** Encounter various obstacles and projectiles. A single hit results in a "Game Over," requiring strategic play.
* **Objective-Based Gameplay:** Players must navigate through the jungle to collect a **coin** in each level to advance.
* **Scoring System:** Track your performance with an integrated point counter.
* **Atmospheric Audio:** Custom-selected background music that enhances the jungle-themed experience.
* **Original Pixel Art:** All character sprites, tilesets, and game assets were custom-designed in a cohesive pixel art style.

## 🚀 Installation & Running

This project is intended for educational purposes and portfolio display. To run the game on an emulator or device:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/KRJEN123/Jungle_Escape.git
    ```
2.  **Open in Android Studio:** Import the project and let Gradle sync all dependencies.
3.  **Build the APK:** Navigate to `Build > Build Bundle(s) / APK(s) > Build APK(s)`.
4.  **Install:** Move the generated `.apk` file to your Android device and install it, or run directly from Android Studio to an attached device/emulator.
